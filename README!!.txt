1.) เข้าลิ้ง " https://github.com/Panucomsci/NuBasicMySQL " กดปุ่ม Clone or Download สีเขียว
2.) ติดตั้งตัวจำลอง web server (ผมแนะนำให้ใช้ xampp โหลดได้ที่ลิ้งนี้ " https://www.apachefriends.org/download.html ")
3.) หลังจากติดตั้ง xampp เปิด xampp กด start ที่ Module " Apache กับ MySQL ทั้งสองอัน "
4.) กด Admin ที่ Module " MySQL " 
5.) ทำการสร้าง database เปล่า ๆ ตั้งชื่อว่า " records " 
6.) ทำการ Import Database โดยเลือกไฟล์ " records " ที่โฟลเดอร์ NusampleSQL
7.) ทำการรันด้วยโปรแกรมพัฒนาตัวใดก็ได้ (แนะนำ dreamweaver)
